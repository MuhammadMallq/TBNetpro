---
description: Menjalankan server dan client chat PyRTC
---

// turbo-all
1. Pergi ke direktori server dan jalankan script server
```powershell
cd server; python server.py
```

2. Buka terminal baru dan jalankan client pertama
```powershell
cd client; python client.py
```

3. Buka terminal baru lagi dan jalankan client kedua untuk simulasi chat
```powershell
cd client; python client.py
```
